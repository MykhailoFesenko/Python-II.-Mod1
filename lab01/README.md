# Lab 01: Names & Objects (binding), mutability, copy, GC

## 1. Description
This laboratory demonstrates that names in Python are bindings to objects, not traditional variables. It covers rebinding vs mutation, argument passing behavior, copy semantics, and reference counting / garbage collection in CPython.
