# Lab 02: Expressions & Control Flow in Python

## 1. Description
This laboratory demonstrates Python's evaluation of truthiness, the distinction between object identity (`is`) and value equality (`==`), and control flow constructs including `if`/`elif` and structural pattern matching (`match`). It also covers the memory-efficient use of comprehensions and lazy evaluation via generators.
