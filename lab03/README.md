# Lab 03: Functions as Objects, Lambda, Closures

## 1. Description
This laboratory explores functional programming paradigms in Python. It demonstrates that functions are first-class objects by passing them as arguments and returning them from factories. It also covers sorting with lambda expressions, state retention using closures, and building efficient data processing pipelines using functional composition (generators, `filter`, and `sum`).

## 2. Environment
* **Python version used:** Python 3.12

## 3. Setup and Run Instructions
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# To run the code:
python src/lab03.py
```
